Belly Button Biodiversity Dashboard

Use the interactive charts below to explore the dataset

Belly Button Biodiversity is a project to study the microbes that live in human navels. The project has collected samples from volunteers and analyzed them to learn more about the microbial diversity of the human body. This dashboard presents some of the data collected in the project in an interactive and easy-to-understand way.
